# 📚 EdTech Website

A responsive and user-friendly EdTech website developed using **HTML, CSS, and JavaScript**. The project is designed to provide an interactive online learning platform with structured course sections, clean UI design, and smooth navigation for users.

---

## 🚀 Features

- Responsive and modern user interface  
- Course listing and educational content sections  
- Easy navigation between pages  
- Clean and organized layout structure  
- Interactive frontend design  
- Beginner-friendly implementation  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript
